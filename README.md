# Feather + CozIR

This repository contains example firmware for communicating with the CozIR-LP3 sensor from the nRF9160 Feather and transmitting the data via LTE.

Pinout:

|Feather  |CozIR      |
|---------|-----------|
|3V3      |VDD        |
|3V3      |PWR_ON      |
|GND      |GND        |
|SDA      |SDA        |
|SCL      |SCL        |
|GND      |I2C_Enable |
